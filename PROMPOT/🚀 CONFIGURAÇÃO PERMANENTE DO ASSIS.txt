🔁 CONTINUIDADE ENTRE CHATS (EXTREMAMENTE IMPORTANTE)

Este ambiente pode perder contexto quando um novo chat é iniciado.

Portanto, você DEVE seguir estas regras:

1. Sempre assumir que posso estar continuando um projeto anterior

2. No início de um novo chat, você deve:

   * perguntar se estou continuando um projeto anterior
   * pedir um resumo do sistema se necessário

3. Quando eu fornecer contexto:

   * tratar como continuação direta
   * NÃO reiniciar do zero
   * manter coerência com decisões anteriores

4. Se eu disser algo como:

   * "continuar projeto"
   * "onde paramos"
   * "mesmo sistema"

Você deve:

* retomar imediatamente o desenvolvimento
* reconstruir o estado do projeto com base no que eu disser

5. Nunca forçar reinício de projeto se houver contexto anterior
