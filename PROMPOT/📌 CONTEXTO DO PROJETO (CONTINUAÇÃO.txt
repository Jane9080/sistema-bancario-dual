📌 CONTEXTO DO PROJETO (CONTINUAÇÃO - NÃO COMEÇAR DO ZERO)

🔹 Nome do projeto:
Sistema Web Bancário Multi-Banco

🔹 Tipo:
Sistema web completo (backend + frontend) com área de cliente e administrador

🔹 Objetivo:
Desenvolver um sistema bancário online que suporta múltiplos bancos independentes, com funcionalidades completas para clientes e administradores.

---

🏦 CONTEXTO FUNCIONAL

O sistema suporta dois bancos:

* Banco Comercial de Kent (tema azul)
* Banco Popular de Jane (tema verde)

Moeda: CVE (Escudo Cabo-verdiano)

Cada banco possui:

* Base de dados própria (SQLite)
* Interface própria
* Gestão independente

---

⚙️ TECNOLOGIAS

* Backend: Node.js + Express
* Banco de dados: SQLite
* Frontend: HTML + CSS + JavaScript (vanilla)
* Arquitetura: MVC modular

---

🧠 HIPÓTESES TÉCNICAS

Possíveis causas:

1. Rota `/admin/change-password` inexistente ou mal definida
2. Middleware de autenticação de admin com falha
3. Controller ou service não executado corretamente
4. Token não está sendo enviado corretamente no frontend
5. Erro no backend retornando HTML em vez de JSON
